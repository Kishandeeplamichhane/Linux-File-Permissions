# Linux File Permissions

## Overview

This project demonstrates how to manage file and directory permissions in Linux using the `ls` and `chmod` commands. The objective was to review existing permissions and update them according to organizational security requirements.

## Objectives

* Check file and directory permissions.
* Understand Linux permission strings.
* Modify file permissions using `chmod`.
* Apply secure access control based on user requirements.
* Verify permission changes.

## Commands Used

* `ls -la`
* `chmod`

## Tasks Performed

* Inspected file and directory permissions using `ls -la`.
* Interpreted Linux permission strings (`rwx`).
* Removed unnecessary write permission from files.
* Updated permissions for a hidden file.
* Modified directory permissions to restrict unauthorized access.
* Verified all permission changes after applying them.

* Linux Administration
* File Permission Management
* Access Control
* Linux Command Line
* System Security
